Database management system (DBMS)

is a system (program) that provides a database interface for storing and retrieving information. We are more interested in software systems rather than manual systems because they can do the job more efficiently. Common features of the DBMS include:

• persistent storage with backup and recovery for reliability

• capacity for large amount of data

• an easy to use interface language (SQL-structured query language)

• efficient retrieval mechanisms

• multi-user support

• security management
