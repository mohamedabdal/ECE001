Types of Databse Systems

Relational database Distributed database
It was invented in the year 1970 AD, in which data is stored in the form of tables that contain at least one column, which expresses the data type, in addition to rows, which are inputs that include specific data, the number of which is the same number of columns (categories) available, for example if the table consists of three Columns, which are the name, age, and length, so the entry "Name: Mohamed, Age: 20, Length: 170" can be added to this table. SQL is usually used in this type of database to deal with the data in it. Databases of this type are distributed in more than one place, i.e. more than one computer device, so that the operations executed on the data are dispersed or repeated in different places of the same network, and such type of databases can be homogeneous; Distributed devices containing the database are identical to each other in specifications (hardware, operating system, etc.), or heterogeneous; The specifications of the devices used differ.
